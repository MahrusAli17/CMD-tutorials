Topic IV - Getting Information using WMIC commands in CMD
--------------------------------------------------------------------------------------------------
1. Initialize WMIC Command
   •	 wmic
   •	 Example: wmic

--------------------------------------------------------------------------------------------------
2. Query CPU Information
   •	 cpu
   •	 Example: wmic:root\cli> cpu

--------------------------------------------------------------------------------------------------
3. Exit WMIC Environment
   •	 quit
   •	 Example: wmic:root\cli> quit

--------------------------------------------------------------------------------------------------
