Topic II - Basic Files and Folders Operations - 
Create Copy Move and Delete Commands

--------------------------------------------------------------------------------------------------
1. Change Directory
   •	cd [path]
   •	Example: cd C:\Users

--------------------------------------------------------------------------------------------------
2. Rename File
   •	ren [old.format] [new.format]
   •	Example: ren old.txt new.txt

--------------------------------------------------------------------------------------------------
3. Copy File
   •	copy [file-name] (Path where you want to paste)
   •	Example: copy one.txt "C:\Users\Mahrus Ali\Desktop"

--------------------------------------------------------------------------------------------------
4. Move File
   •	move [file-name] (Path where you want to paste)
   •	Example: move one.txt "C:\Users\Mahrus Ali\Desktop"

--------------------------------------------------------------------------------------------------
5. Delete Single File
   •	del [file.format]
   •	Example: del one.txt

--------------------------------------------------------------------------------------------------
6. Delete Multiple Files
   •	del -.format
   •	Example: del -.txt

--------------------------------------------------------------------------------------------------
7. Create File with Text
   •	echo [text] > [file-name]
   •	Example: echo This is a sample text > file.txt

--------------------------------------------------------------------------------------------------
8. Append Text to File
   •	echo [text] >> [file-name]
   •	Example: echo This is also crazy sample text >> file.txt

--------------------------------------------------------------------------------------------------
9. Create File and Input Text from Console
   •	copy con [file-name]
   •	Example: copy con example.txt
